Sistema de Gestión de Garage

Proyecto Java para administrar un sitemas de garage.

Descripción

Este proyecto desarrolla un sistema en Java que permite gestionar un garage.

El sistema administra el ingreso, permanencia y salida de vehículos.

Conceptos aplicados

● correcto uso de Programación Orientada a Objetos
● implementación de herencia y polimorfismo
● uso adecuado de interfaces
● manejo de colecciones
● validaciones de negocio
● manejo de excepciones
● funcionamiento del sistema
● claridad y organización del código
● documentación UML
● uso de control de versiones
